What Is Computer Structure?

🏙️ Real World Analogy

Think of a computer like a city. The city has different districts (components) that each have a job — the financial district (CPU) makes decisions, the warehouses (Memory) store things temporarily, the roads (Buses) move stuff around, and the ports/airports (I/O) bring things in and out. Structure is the map of how all these districts are laid out and connected.

Structure refers to the way in which the components of a computer are interrelated. It's the physical and logical arrangement — the blueprint of the machine.


A computer has four main structural components:

  • CPU (Central Processing Unit) — controls the operation of the computer and performs data processing
  • Main Memory — stores data and programs currently being used
  • I/O (Input/Output) — transfers data between the computer and the external environment
  • System Interconnection — the mechanism(s) that provides communication among CPU, memory, and I/O (usually a bus)

What Is Computer Function?

⚙️ Real World Analogy

If Structure is the city map, then Function is what happens in the city day-to-day — the flow of traffic, transactions, the processes running. Function is about what the computer does, not just how it's built.

All computers perform the same four basic functions. Click each to explore:

🔢 Data Processing

The computer must be able to process data. The data may take a wide variety of forms, and the range of processing requirements is broad. You write code → CPU executes it. You add two numbers → CPU processes that. This is the core purpose of the CPU — it transforms data.

💾 Data Storage

There are two levels of storage. Short-term: even if there's no need to process data immediately, the data must still be stored in the computer (RAM holds it temporarily). Long-term: data files need to be stored even when the computer is off — that's your hard drive or SSD. Storage is the computer's memory and filing cabinet.

↔️ Data Movement

The computer must be able to move data between itself and the outside world. When data are received from or delivered to a device directly connected to the computer — this is called I/O. When data are moved over longer distances — this is called data communications. Think: typing on a keyboard, printing a document, browsing the internet.

🎛️ Control

There must be control of the three previous functions. A special unit within the CPU — the Control Unit — manages and coordinates the resources and activities. It's like a traffic controller — directing when to read memory, when to write to disk, when to accept keyboard input. Without control, the computer would be chaos.

Structure vs Function — The Key Distinction

This is a crucial distinction that Stallings (your textbook author) emphasises heavily:


  • Structure = How components are connected (the static layout)
  • Function = What each component does (the dynamic behaviour)

Both are needed to fully understand a computer. You can't understand a car by only knowing where the parts are placed — you also need to know what each part does.

🧬 Think of it like the Human Body

Structure: Heart, lungs, brain, arteries — their positions and connections.

Function: Heart pumps blood, lungs exchange oxygen, brain processes signals.

Knowing anatomy (structure) without physiology (function) — or vice versa — gives you an incomplete picture. Computer architecture is the same way.

🔥 Challenge Quiz — Test Yourself

Q1. When you're playing a video game and your character moves, which computer function is primarily happening?

✅ Data Processing! The CPU is constantly processing your inputs (keyboard/controller) and calculating physics, animations, and game logic to produce output on screen.

Q2. Your friend says "Structure and Function are basically the same thing." What's wrong with this?

✅ Exactly right! Structure = static arrangement/connections. Function = dynamic behaviour. Both are needed for a complete picture — one describes the map, the other describes the journey.

Q3. Which of the four structural components of a computer acts as the "highway" connecting everything?

✅ The System Bus! It's the communication highway. Without it, the CPU, Memory, and I/O couldn't exchange any data — they'd be isolated islands. Data, addresses, and control signals all travel on the bus.
